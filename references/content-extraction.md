# 网页内容抓取

从目标网页提取结构化内容，用于生成视频脚本。

## 方法

使用 Hermes 的 `delegate_task` 派生子代理，通过浏览器工具获取：

```
delegate_task(
  goal="抓取网页 <URL> 的完整文字内容，按标题层级结构化输出",
  toolsets=["browser", "web"]
)
```

子代理执行步骤：
1. `browser_navigate(url)` → 加载页面
2. `browser_snapshot(full=true)` → 获取完整 DOM 文本
3. 如需更精确提取，用 `browser_console(expression="...")` 执行 JS 提取特定元素

## 提取后数据处理

### 内容归纳
- H1 → 视频主标题
- H2 各章节标题 → 场景主题
- 功能列表/要点 → 场景卡片内容
- 段落描述 → 旁白文案素材

### 关键数据筛选
从文章中找出可用于"数据/证据"场景的数字：
- 百分比（增速、降低、占比…）
- 绝对值（案例数、用户数、金额…）
- 时间数据（缩短/节省的时间…）

### 联系信息
CTA 场景需要：
- 公司/品牌名
- 电话
- 网址
- 地址（如有）

## 非网页内容源

如果是文档、PDF、纯文本而非网页：
- PDF：用 `ocr-and-documents` 技能提取文本
- 纯文本：直接作为 material 传给脚本设计阶段
- 用户口述：整理成结构化提纲后再进入脚本设计
